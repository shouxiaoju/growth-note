# DESIGN.md

## 气质与意象
清爽学术书房——晨光透过白色纱帘洒在极简书房，书架整齐分门别类，笔记本摊开在木桌上，空气中漂浮着微尘。专注阅读、无干扰、有条理。

## 视觉策略
- 参考菜鸟教程的清晰分类 + MDN 的文档结构 + 阮一峰博客的个人风格
- 图形语言：几何线条分隔，圆角卡片，代码块精致
- 留白充足，信息密度适中

## 配色方案
### 浅色模式
- 主色：#2563EB（蓝色，专业可靠）— 源自晴空蓝天
- 辅助色：#7C3AED（紫色，AI 科技感）— 源自暮色霞光
- 背景：#FFFFFF
- 正文：#1F2937
- 代码背景：#F8FAFC
- 边框：#E5E7EB

### 暗色模式
- 主色：#60A5FA
- 辅助色：#A78BFA
- 背景：#0F172A
- 正文：#E2E8F0
- 代码背景：#1E293B
- 边框：#334155

## 字体排版
- 正文：系统字体栈（PingFang SC, Microsoft YaHei, sans-serif）
- 代码：系统等宽字体栈（SF Mono, Menlo, Monaco, Consolas）
- 标题：系统字体加粗，不额外引入字体
- 排版节奏：正文 16px，标题分级 32/24/20/18px

## 动效与交互
- 运动性格：沉稳、克制，不过度花哨
- 页面过渡：fade-in 渐入
- 悬停：卡片微上浮 + 阴影加深
- 侧边栏：平滑展开/收起
- 搜索框：Command+K 快捷键唤起，模态弹出

## 页面结构
- 顶部导航栏：Logo + 搜索 + 分类导航 + 暗黑模式切换
- 文档页：左侧分类目录（可折叠）+ 中间正文 + 右侧 TOC
- 首页：Hero 大标题 + 最近更新卡片 + 热门笔记 + 分类入口
- 关于页：个人介绍 + 技术栈标签 + 联系方式

## 响应式
- 桌面端 ≥1024px：三栏（侧边栏 + 内容 + TOC）
- 平板端 768-1023px：侧边栏可折叠，TOC 隐藏
- 移动端 <768px：汉堡菜单 + 全屏内容

## 设计禁忌
- 禁止花哨渐变背景和闪亮动效
- 禁止信息过载——每屏聚焦一个主题
- 禁止冰冷的模板感——保持个人温度
- 禁止代码块无语法高亮
