# JavaScript 基础语法

> JavaScript 是 Web 的编程语言，掌握基础语法是前端开发的核心。

## 1. 变量声明

ES6 引入了 `let` 和 `const`，替代传统的 `var`：

```javascript
// ❌ 避免使用 var（函数作用域，变量提升）
var name = 'old';

// ✅ let - 可重新赋值的变量（块级作用域）
let count = 0;
count = 1;

// ✅ const - 不可重新赋值的常量（块级作用域）
const PI = 3.14159;
```

> 💡 个人分析：默认使用 `const`，只有需要重新赋值时才用 `let`，永远不用 `var`。

## 2. 数据类型

JavaScript 有 7 种原始类型 + 1 种引用类型：

| 类型 | 示例 | typeof |
|------|------|--------|
| Number | `42`, `3.14` | `'number'` |
| String | `'hello'` | `'string'` |
| Boolean | `true`, `false` | `'boolean'` |
| Undefined | `undefined` | `'undefined'` |
| Null | `null` | `'object'` ⚠️ |
| Symbol | `Symbol()` | `'symbol'` |
| BigInt | `9007199254740991n` | `'bigint'` |
| Object | `{}`, `[]` | `'object'` |

```javascript
// 类型判断
typeof 42          // 'number'
typeof 'hello'     // 'string'
typeof null        // 'object' ⚠️ 历史遗留 bug
Array.isArray([1]) // true
```

## 3. 运算符

```javascript
// 等值比较
===  // 严格相等（类型 + 值），推荐
==   // 宽松相等（会类型转换），避免使用

// 空值合并
const value = null ?? 'default'; // 'default'

// 可选链
const name = user?.profile?.name; // 安全访问

// 解构赋值
const { name, age } = person;
const [first, ...rest] = array;
```

## 4. 流程控制

```javascript
// if-else
if (score >= 90) {
  grade = 'A';
} else if (score >= 80) {
  grade = 'B';
} else {
  grade = 'C';
}

// for...of（遍历可迭代对象）
for (const item of array) {
  console.log(item);
}

// for...in（遍历对象键名）
for (const key in object) {
  console.log(key, object[key]);
}
```

## 5. 个人分析

- JavaScript 是弱类型语言，`===` 严格比较能避免大部分类型陷阱
- 现代开发中优先使用 `for...of` 和数组方法（`map/filter/reduce`）
- 可选链 `?.` 和空值合并 `??` 大幅简化了防御性编程
