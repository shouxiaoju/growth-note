# 微信小程序生命周期

> 理解生命周期是开发小程序的第一步，它决定了代码在什么时候执行。

## 1. 页面生命周期

```javascript
Page({
  onLoad(options) {
    // 页面加载时触发，只执行一次
    console.log('页面参数:', options);
  },
  onShow() {
    // 页面显示/切入前台时触发
  },
  onReady() {
    // 页面初次渲染完成，只执行一次
  },
  onHide() {
    // 页面隐藏/切入后台时触发
  },
  onUnload() {
    // 页面卸载时触发
  },
});
```

## 2. 生命周期执行顺序

```
 onLoad → onShow → onReady
                     ↑
 onShow ←────── 页面再次显示
```

## 3. 个人分析

- `onLoad` 适合获取页面参数和初始化数据
- `onShow` 适合刷新数据（如从其他页面返回时）
- `onReady` 适合获取节点信息（如 wx.createSelectorQuery）
