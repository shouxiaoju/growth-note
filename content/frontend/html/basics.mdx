# HTML5 基础语法

> 一句话概述：HTML 是网页的骨架，掌握基础语法是前端开发的第一步。

## 1. HTML 文档结构

每个 HTML5 文档都遵循相同的基本结构：

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>页面标题</title>
</head>
<body>
  <!-- 页面内容 -->
</body>
</html>
```

> 💡 个人分析：`<!DOCTYPE html>` 声明不是 HTML 标签，它是指示浏览器使用 HTML5 标准模式渲染页面的指令。

## 2. 常用标签

### 标题标签

```html
<h1>一级标题</h1>
<h2>二级标题</h2>
<h3>三级标题</h3>
```

### 文本标签

- `<p>` 段落
- `<strong>` 加粗（语义强调）
- `<em>` 斜体（语义强调）
- `<span>` 行内容器

### 链接与图片

```html
<a href="https://example.com" target="_blank">链接文本</a>
<img src="image.jpg" alt="图片描述" width="300">
```

> ⚠️ 易错点：`<img>` 标签必须添加 `alt` 属性，这对无障碍访问和 SEO 至关重要。

## 3. 语义化标签（HTML5 新增）

HTML5 引入了一系列语义化标签，让页面结构更有意义：

| 标签 | 用途 |
|------|------|
| `<header>` | 页面或区块头部 |
| `<nav>` | 导航区域 |
| `<main>` | 主要内容 |
| `<article>` | 独立文章 |
| `<section>` | 内容分区 |
| `<aside>` | 侧边栏 |
| `<footer>` | 页脚 |

```html
<header>
  <nav>导航链接</nav>
</header>
<main>
  <article>
    <h1>文章标题</h1>
    <p>文章内容...</p>
  </article>
  <aside>侧边内容</aside>
</main>
<footer>版权信息</footer>
```

## 4. 个人分析

- 语义化标签不仅提升可读性，还显著改善 SEO 和无障碍访问
- 实际开发中，优先使用语义化标签而非 `<div>` 嵌套
- `<h1>` 每个页面只应使用一个，用于主标题
