# CSS3 Flexbox 布局完全指南

> Flexbox 是 CSS3 最强大的布局模式之一，一维布局的首选方案。

## 1. 基本概念

Flexbox 布局由容器（flex container）和项目（flex item）组成：

```css
.container {
  display: flex;
  /* 或 inline-flex */
}
```

设置 `display: flex` 后，容器成为弹性容器，直接子元素成为弹性项目。

## 2. 容器属性

### flex-direction（主轴方向）

```css
.container {
  flex-direction: row;            /* 默认，水平从左到右 */
  flex-direction: row-reverse;    /* 水平从右到左 */
  flex-direction: column;         /* 垂直从上到下 */
  flex-direction: column-reverse; /* 垂直从下到上 */
}
```

### justify-content（主轴对齐）

```css
.container {
  justify-content: flex-start;    /* 默认，起点对齐 */
  justify-content: center;        /* 居中 */
  justify-content: space-between; /* 两端对齐，项目间等距 */
  justify-content: space-around;  /* 项目两侧等距 */
  justify-content: space-evenly;  /* 完全均匀分布 */
}
```

### align-items（交叉轴对齐）

```css
.container {
  align-items: stretch;    /* 默认，拉伸填满 */
  align-items: flex-start; /* 起点对齐 */
  align-items: center;     /* 居中 */
  align-items: baseline;   /* 文字基线对齐 */
}
```

## 3. 项目属性

### flex 简写

```css
.item {
  flex: 1;           /* flex-grow: 1, flex-shrink: 1, flex-basis: 0% */
  flex: auto;        /* flex-grow: 1, flex-shrink: 1, flex-basis: auto */
  flex: none;        /* flex-grow: 0, flex-shrink: 0, flex-basis: auto */
  flex: 0 0 200px;   /* 固定 200px，不伸缩 */
}
```

> 💡 个人分析：推荐优先使用 `flex` 简写而非单独设置 `flex-grow/shrink/basis`，简写能避免常见陷阱。

## 4. 常见布局模式

### 水平垂直居中

```css
.center-container {
  display: flex;
  justify-content: center;
  align-items: center;
  height: 100vh;
}
```

### 等分布局

```css
.equal-columns {
  display: flex;
}
.equal-columns > * {
  flex: 1;
}
```

### 侧边栏固定 + 内容自适应

```css
.layout {
  display: flex;
}
.sidebar {
  flex: 0 0 250px;
}
.content {
  flex: 1;
}
```

## 5. 个人分析

- Flexbox 解决了 90% 的一维布局问题，Grid 解决二维布局
- 实际开发中，最常见的组合是 `display: flex` + `flex: 1` + `gap`
- `gap` 属性比 margin 更优雅地处理项目间距
