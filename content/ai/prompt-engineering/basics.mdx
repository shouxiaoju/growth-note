# Prompt 工程基础

> Prompt 工程是与大语言模型高效协作的关键技能，好的提示词能让 AI 输出质量提升数倍。

## 1. 核心原则

### 清晰具体

```text
❌ 帮我写个函数
✅ 用 TypeScript 写一个函数，接收数字数组，返回去重后的升序数组，要求时间复杂度 O(n)
```

### 角色设定

```text
你是一位资深前端架构师，擅长 React 和 TypeScript。
请从架构角度分析以下代码的问题，并给出优化方案。
```

### 分步骤

```text
请按以下步骤分析：
1. 列出代码中的问题
2. 给出每个问题的原因
3. 提供修复方案
4. 给出优化后的完整代码
```

## 2. 常用技巧

| 技巧 | 说明 | 示例 |
|------|------|------|
| Few-shot | 给出示例 | 输入：... 输出：... |
| Chain-of-Thought | 思维链 | 请一步步思考 |
| 自我反思 | 检查输出 | 请检查你的回答是否准确 |
| 格式约束 | 指定格式 | 请以 JSON 格式输出 |

## 3. 个人分析

- Prompt 工程不是"调参数"，而是"设计沟通方式"
- 最有效的技巧是 Few-shot + 格式约束的组合
- 好的 Prompt 是可复用的，建议建立个人 Prompt 库
