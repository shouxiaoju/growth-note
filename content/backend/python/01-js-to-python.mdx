# JavaScript To Python
>作为一个前端开发者，已经掌握了js，学习Python不是从零开始，而是换一种语法表达你已经理解的概念。

## 命名规范

**相同点：**
- 第一个字符必须以字母（a-z, A-Z）或下划线
- 大小写敏感
- 禁止使用保留关键字，如 if、for、class 等不能作为标识符。

**不同点：**
- Python 使用下划线分隔单词，如 snake_case而js使用驼峰命名法，如 camelCase

|特性|JavaScript|Python|
|:-------|:--------|:--------|
|普通变量/函数|camelCase (小驼峰命名法)|snake_case (蛇形命名法)|
|类名 (Class)|PascalCase (大驼峰)|PascalCase (大驼峰)|
|常量 (Constant)|UPPER_SNAKE_CASE (常见)|UPPER_SNAKE_CASE|

## 注释
** JavaScript **
- 单行注释以 // 开头。
- 多行注释以 /* 开始，以 */ 结尾。
```JavaScript
// 输出标题：
document.getElementById("myH1").innerHTML="欢迎来到我的主页";
/*
下面的这些代码会输出
一个标题和一个段落
并将代表主页的开始
*/
document.getElementById("myH1").innerHTML="欢迎来到我的主页";
document.getElementById("myP").innerHTML="这是我的第一个段落。";
```
** Python **
- 单行注释以 # 开头
- 多行字符串（由三个单引号 ''' 或三个双引号 """ 包围的文本块）可以用作多行注释。
```python
# 这是一个注释
print("Hello, World!")
'''
这是多行注释，用三个单引号
这是多行注释，用三个单引号
这是多行注释，用三个单引号
'''
print("Hello, World!")

"""
这也是多行注释，用三个双引号
这也是多行注释，用三个双引号
这也是多行注释，用三个双引号
"""
# 多行注释是由三个单引号 ''' 或三个双引号 """ 来定义的，这种注释方式不能嵌套使用。
```
> ** 注意: **虽然多行字符串在这里被当作多行注释使用，但它实际上是一个字符串，我们只要不使用它，它不会影响程序的运行。
>这些字符串在代码中可以被放置在一些位置，而不引起实际的执行，从而达到注释的效果。

### Docstring 文档字符串
Python 的 Docstring（文档字符串）是一种特殊的注释，用于为函数、类、模块等添加文档说明。与普通注释不同，Docstring 可以通过 __doc__ 属性直接访问，也可以使用 help() 函数查看。 
Docstring 使用三个双引号 """ 或三个单引号 ''' 包围，放在函数、类、模块的开头。
```python
def add(a, b):
    """返回两数之和"""
    return a + b

# 通过 __doc__ 属性访问
print(add.__do c__)  # 输出: 返回两数之和

# 使用 help() 函数
help(add)
'''Help on function add in module __main__:

add(a, b)
    返回两数之和
'''
```

## 数据类型
>  核心差异：JS 只有 8 种类型（7 原始 + Object），Python 有丰富的内置类型体系。JS 的 number 一把梭，Python 拆成 int + float + complex；JS 的 Object 包打天下，Python 拆成 dict + tuple + set + 自定义类。

|分类|JS|Python|关键差异|
|:-------|:-------|:-------|:-------|
|整数|number（IEEE 754 双精度）|int（任意精度）|JS 有精度上限，Python 无限大|
|浮点型|number（同上）|float（IEEE 754 双精度）|浮点精度问题两者都有|
|字符串|string|str|Python字符串用单双引号都可以，没有反引号模板字符串|
|布尔|boolean|bool|Python是True/False（大写）表示|
|空值|null + undefined|None|JS 有两个空值，Python 只有一个|
|数组|Array|list|Python列表，方括号[1,2,3]|
|对象/字典|Object|dict|Python字典，花括号\{"key": "value"}|
|集合|Set| set|Python集合，花括号{1,2,3}（没有冒号区分dict）|
|映射|Map|map|Python的dict就是有序的，不需要单独的Map
|Symbol|Symbol|❌没有|——|
|复数|❌ 没有|complex（内置）|Python 原生支持 3+4j

except json.JSONDecodeError as e:     # 捕获特定类型
    print(e)
except Exception as e:                # 捕获所有

finally:
    cleanup()
raise ValueError("出错了")            # raise 代替 throw

## 列表推导式 —— 必须掌握的 Pythonic 写法
JS 完全没有对应物，这是 Python 最强大的语法特性。
```python
# 基本形式：[表达式 for 变量 in 可迭代对象 if 条件]

# 映射（= map）
[x * 2 for x in range(5)]                    # [0, 2, 4, 6, 8]

# 过滤（= filter）
[x for x in range(10) if x % 2 == 0]         # [0, 2, 4, 6, 8]

# 映射 + 过滤
[x ** 2 for x in range(10) if x % 2 == 0]   # [0, 4, 16, 36, 64]

# 嵌套循环（= flatMap）
[(x, y) for x in range(3) for y in range(3)]

# 字典推导式
{name: len(name) for name in ["Python", "JS", "Go"]}
# {"Python": 6, "JS": 2, "Go": 2}

# 集合推导式
{x % 3 for x in range(10)}                   # {0, 1, 2}

# 三元表达式结合
["even" if x % 2 == 0 else "odd" for x in range(5)]

```

## 避坑指南 —— JS 开发者最常踩的 10 个坑
1. 缩进是语法：混用 Tab 和空格报 IndentationError，统一用 4 空格
2. **没有 ++/--**：i++ 是语法错误，用 i += 1
3. 字典没有点语法：obj.name 报错，必须 obj["name"]
4. **True/False/None 大写**：小写报 NameError
5. **and/or/not**：不能用 &&/||/!
6. **elif 不是 else if**：一个词，中间没空格
7. **range 不含终点**：range(5) 是 0-4，和 for(i=0;i< 5;i++) 一致
8. **整除用 //**：7/2 是 3.5，7//2 才是 3
9. **self 必须显式写**：类方法第一个参数永远是 self
10. **asyncio.run() 别忘**：async 函数不会自动执行
